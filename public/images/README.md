# Папка для изображений

## Структура папок:

```
public/
└── images/
    ├── categories/
    │   ├── converter.png          # Частотные преобразователи
    │   ├── motor.png              # Электродвигатели
    │   ├── equipment.png          # Допоборудование к ПЧ и УПП
    │   └── soft-starter.png       # Устройства плавного пуска
    └── callback_bg.png            # Фоновая картинка для формы обратной связи
```

## Как загрузить изображения:

### Категории товаров:
1. Создай папку `public/images/categories/` (если её нет)
2. Загрузи изображения с такими именами:
   - `converter.png` - для частотных преобразователей
   - `motor.png` - для электродвигателей
   - `equipment.png` - для допоборудования
   - `soft-starter.png` - для устройств плавного пуска

### Фоновая картинка для формы:
1. Загрузи файл `callback_bg.png` в папку `public/images/`
2. Путь к файлу: `public/images/callback_bg.png`
3. В коде используется путь: `/images/callback_bg.png`

## Формат изображений:

- Рекомендуемый формат: PNG или JPG
- Размер категорий: примерно 400x400px или больше (квадратные)
- Размер фона формы: рекомендуется ширина 800-1200px, высота 600-800px
- Оптимизируй изображения перед загрузкой для быстрой загрузки

## Использование в коде:

Изображения из папки `public` доступны по пути начиная с `/`:

```tsx
// Правильно:
<Image src="/images/categories/converter.png" ... />
<div style={{ backgroundImage: 'url(/images/callback_bg.png)' }} />

// Неправильно:
<Image src="/public/images/categories/converter.png" ... />
```

Next.js автоматически оптимизирует изображения через компонент `Image`.
